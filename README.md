# Тестирование шифрования RSA (RsaCipher Tests)

## Описание проекта
В данном проекте реализовано модульное тестирование класса RsaCipher с использованием фреймворка MSTest в среде разработки Microsoft Visual Studio.

Приложение реализует упрощённый алгоритм RSA и включает:

генерацию ключей (GenerateKeys)
шифрование текста (Encrypt)
дешифрование текста (Decrypt)

Также проверяется корректность обработки исключений при неверных входных данных.

# Реализованные тесты
1. GenerateKeys_ValidPrimes_CalculatesCorrectParameters

Проверяет корректность вычисления параметров RSA:

модуля N
открытой экспоненты E
закрытого ключа D
2. EncryptAndDecrypt_NormalString_ReturnsOriginalText

Проверяет обратимость шифрования:
после шифрования и дешифрования строка должна остаться неизменной.

3. EncryptAndDecrypt_CyrillicAndSymbols_ReturnsOriginalText

Проверяет корректную работу с:

кириллицей
специальными символами

4. Encrypt_EmptyString_ThrowsArgumentException

Проверяет выброс исключения ArgumentException при попытке зашифровать пустую строку.

5. Encrypt_String_ResultIsDifferentFromInput

Проверяет, что зашифрованные данные отличаются от исходных.

6. Decrypt_NullArray_ThrowsArgumentException

Проверяет выброс исключения при попытке дешифровать null.
